How to Use Sidney Apparels OPS

×

Getting Started

1

Add Operators - Start by adding operators in the Operators tab. Each operator needs an ID, name, and current sewing line.

Tip: You can import operators from Excel using the Import Excel button.

2

Record Performance - Go to the Performance Data tab to add manual performance records or use the Time Study tab for live measurements.

Tip: Time Study automatically calculates efficiency with 11% allowance.

Using the Time Study

1

Select Operator - Choose an operator from the dropdown list.

2

Enter Operation Details - Fill in line number, style, product description, operation name, and machine.

3

Record Cycle Times - Click Start to begin timing. Click Lap after each cycle (3 or 5 cycles recommended).

4

Save Results - The system automatically calculates working SMV and efficiency. Click Save to record.

Analysis Features

1

Performance Analysis - Click "Analyze Performance" to view charts and statistics.

2

Line & Machine Analysis - Check how many A/B/C skill operators are on specific lines and machines.

3

Garment SMV by Line - Select a line in the Performance tab to see average standard vs working SMV.

Pro Tips

Skill Levels - Operators are automatically grouped into A, B, or C based on their performance records.

Export Data - Use the Export Excel buttons to backup your data or share reports.

Real-time Updates - All data syncs automatically. The clock shows the last sync time.
